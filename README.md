# \# 🖥️ Project Aura: Home Server Infrastructure



#### Welcome to the configuration repository for \*\*Aura\*\*, my personal home server. This repository houses the deployment files, Docker setups, and maintenance scripts I use to manage my self-hosted environment and business applications.



### \## ⚙️ Core Architecture

**\* \*\*OS:\*\* Ubuntu Server**

**\* \*\*Management Dashboard:\*\* CasaOS**

**\* \*\*Containerization:\*\* Docker \& Docker Compose**

**\* \*\*Networking/VPN:\*\* Tailscale**

**\* \*\*Reverse Proxy:\*\* Nginx Proxy Manager (NPM)**



### \## 📦 Hosted Services \& Projects

**Here are the key services and custom projects currently running on Aura:**



###### \* \*\*Open Claw:\*\* A B2B lead generation engine. This setup also handles automated content distribution by publishing blog pages directly to Hashnode.

###### \* \*\*Immich:\*\* High-performance, self-hosted photo and video backup.

###### \* \*\*Uptime Kuma:\*\* Self-hosted monitoring tool for keeping track of service health.



### \## 🔒 Security \& Access

**\* Access to the server network is secured via \*\*Tailscale\*\*.**

**\* Public-facing services and APIs are routed securely through \*\*Nginx Proxy Manager\*\* with SSL certificates.**



### \## 📂 Repository Structure

**\* `/docker-compose/` - Contains the `.yml` files for spinning up individual services.**

**\* `/scripts/` - Bash scripts for automated backups and system updates.**
